# 간단한 자기소개 웹페이지 제작
- 개발언어: HTML, CSS
- 개발환경: EditPlus
- 소개
  - FTP접속 및 관리를 도와주는 프로그램 이용
  - 무료 호스팅 도메인 등록 후 서버 연결
  - 메뉴 클릭 시 다음 페이지 이동
  - HTML의 다양한 태그들 이용
  - CSS border-style Property 이용
